## ğŸ—ï¸ Architecture & Workflow

The platform follows a streamlined user flow to ensure maximum engagement and learning retention: 

1. **User Login:** Access via Web/App UI using Firebase Auth.
2. **Activity Selection:** Choose from the Coding Playground, Quizzes, or Projects.
3. **AI Analysis Layer:** User code is sent to **Gemini 2.5 Flash** for analysis.
4. **Feedback Loop:** Users receive line-by-line explanations and debugging hints.
5. **Progress Storage:** Data and analytics are updated in **Firebase Firestore**.
6. **Adaptive Update:** The user's learning level adjusts based on performance.
